OpenClaw 安装配置指南

这篇文章记录在 NAS 上搭建 OpenClaw 的最小流程，适合先把服务跑起来，再慢慢补渠道和模型配置。

环境准备

  • NAS 一台，本文示例是飞牛 NAS
  • Docker
  • 可用的模型 API Key，例如 OpenAI、MiniMax 等

安装步骤

  1. 在飞牛 NAS 的应用中心安装 Docker。
  2. 拉取镜像：
docker pull openclaw/openclaw
  1. 运行容器：
docker run -d \
  --name openclaw \
  -p 8080:8080 \
  -v ~/.openclaw:/data \
  openclaw/openclaw
  1. 编辑 ~/.openclaw/config.yaml，填写 API 与渠道参数。

接入飞书

  1. 创建飞书应用
  2. 获取 App ID 与 App Secret
  3. 把参数写入配置文件
  4. 把机器人加进群聊
