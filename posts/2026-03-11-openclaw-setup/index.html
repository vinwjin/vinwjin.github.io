OpenClaw 安装配置指南

发表于2026-03-11|AI教程

简介

OpenClaw 是一个运行在 NAS 上的 AI 助手框架，支持多种渠道接入（飞书、Telegram 等）。

环境准备

  • NAS 一台（飞牛NAS）
  • Docker
  • API Key（MiniMax、OpenAI 等）

安装步骤

1. 安装 Docker

在飞牛NAS 的应用中心安装 Docker 套件。

2. 拉取镜像

3. 运行容器

4. 配置

编辑 ~/.openclaw/config.yaml，配置你的 AI API 和渠道。

接入飞书

  1. 创建飞书应用
  2. 获取 App ID 和 App Secret
  3. 在配置文件中填写
  4. 添加机器人到群聊
