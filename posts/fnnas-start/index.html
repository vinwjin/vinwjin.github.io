飞牛NAS 使用初体验

一直想搭一个自己的私有云，能存文件、跑服务，也能顺手折腾一点 AI 工具。飞牛 NAS 的上手门槛相对低，所以先从它开始。

为什么选飞牛NAS

  • 国产系统，本土化做得不错
  • 价格相对友好
  • Docker 支持到位
  • 界面简洁，适合先搭起来

初步感受

  1. 安装简单，和普通软件差不多
  2. 界面清爽，容易上手
  3. Docker 好用，常见服务都能跑
  4. 性价比高，适合个人折腾

待改进的地方

  • 应用生态还在完善
  • 更专业的功能和群晖相比还有差距

目前已经跑起来的内容

  • OpenClaw AI 助手
  • 文件存储
  • 一些常用 Docker 容器
