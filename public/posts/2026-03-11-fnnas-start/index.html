飞牛NAS 使用初体验

发表于2026-03-11|NAS体验
|浏览量:

为什么要用 NAS

一直想有个自己的私有云，存照片、跑服务、折腾点东西。

为什么选飞牛NAS

  • 🐂 国产系统，本土化做得好
  • 💰 价格实惠
  • 🐳 Docker 支持
  • 🖥️ 界面简洁

初步使用感受

优点

  1. 安装简单 - 跟装普通软件一样
  2. 界面清爽 - 容易上手
  3. Docker 好用 - 基本服务都能跑
  4. 性价比高 - 够用不贵

待改进

  • 应用生态还在完善
  • 某些专业功能不如群晖

跑起来了什么

  • 🤖 OpenClaw AI 助手
  • 📚 文件存储
  • 🐧 各种 Docker 容器

未完待续
